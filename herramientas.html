HERRAMIENTAS CONFIABLES PROYECTOS IMBATIBLES

En la exploración del conocimiento, las herramientas adecuadas son tan esenciales como la brújula para un explorador. En el proyecto ARFIS, emprendimos un viaje de aprendizaje para comprender el Movimiento Rectilíneo Uniforme (MRU) a través de la robótica y la metodología STEAM. Para guiarnos en este camino, nos equipamos con un conjunto de herramientas cuidadosamente seleccionadas, cada una con un rol específico para impulsar nuestro avance.

KIT DE COMPONENTES

Microcontrolador Arduino

Un microcontrolador de Arduino es un pequeño ordenador en un chip que ejecuta instrucciones para controlar dispositivos electrónicos. La placa Arduino proporciona interfaces fáciles de usar y un entorno de desarrollo para programar el microcontrolador, popular en proyectos de electrónica y robótica por su accesibilidad y versatilidad.

Sensor SR04

El sensor SR04 es un dispositivo de medición de distancia que utiliza el principio de eco de ultrasonido para calcular la distancia entre el sensor y un objeto. Consiste en un transmisor ultrasónico (emisor) y un receptor ultrasónico (receptor) que operan en conjunto para determinar la distancia.

Motor de Corriente Continua (DC)

Un motor de corriente continua convierte la energía eléctrica en movimiento mecánico. Funciona mediante la aplicación de un voltaje a través de sus terminales, lo que crea un campo magnético que hace girar el eje del motor. Los motores DC se utilizan ampliamente en una variedad de aplicaciones, desde juguetes hasta sistemas de control de precisión.

Protoboard

Un protoboard es una herramienta de prototipado electrónico que permite ensamblar circuitos temporales sin soldadura. Consiste en una placa con orificios conectados eléctricamente entre sí en filas y columnas, facilitando la conexión de componentes electrónicos mediante cables o pines. Es fundamental en el desarrollo y prueba de circuitos antes de su implementación definitiva.

Dupont Jumpers

Los cables Dupont, también conocidos como "Dupont jumpers" o "jumper wires," son cables de conexión ampliamente utilizados en electrónica, especialmente en prototipos y proyectos de desarrollo. Estos cables son esenciales para conectar componentes en placas de prueba (breadboards), módulos, sensores, microcontroladores y otros dispositivos electrónicos.

Chasís

Un chasís de carrito Arduino es la estructura base utilizada para construir un robot móvil. Generalmente incluye una plataforma con espacio para montar componentes como motores, ruedas, baterías y el microcontrolador Arduino. Facilita la creación de proyectos de robótica, permitiendo añadir sensores y otros dispositivos para crear vehículos autónomos o controlados remotamente.

Módulo Bluetooth HC05

El HC-05 es un módulo Bluetooth popular para Arduino. Permite la comunicación inalámbrica entre dispositivos, como controlar un Arduino desde un smartphone, usando protocolos como UART. Es fácil de usar y ampliamente utilizado en proyectos de robótica y domótica.

App móvil

Una app para controlar un carrito Arduino, conectada por Bluetooth o Wi-Fi, permite manejar el robot móvil desde un smartphone, controlando dirección, velocidad y funciones adicionales a través de una interfaz intuitiva.

KIT DE DESARROLLO

Para complementar nuestro kit de herramientas, también utilizamos un kit de desarrollo que nos facilitó la construcción y programación del carro robotizado. Para los estudiantes que van a emplear el recurso inclumos el kit como sofware preprogramado para mayor fascilidad de uso.
Este kit incluía:

Codigo Preprogramado

          
        
#include <SoftwareSerial.h>
#include <Servo.h>

// CONEXIONES PARA EL BLUETOOTH.
const int bluetoothTx = 2;
const int bluetoothRx = 3;

SoftwareSerial bluetooth(bluetoothTx, bluetoothRx);

// MOTOR 1.
const int Motor1A = 5;
const int Motor1B = 6;

// MOTOR 2.
const int Motor2A = 9;
const int Motor2B = 10;

void setup() {
    bluetooth.begin(115200);
    bluetooth.print("$$$");
    delay(100);
    bluetooth.println("U,9600,N");
    bluetooth.begin(9600);

    pinMode(Motor1A, OUTPUT);
    pinMode(Motor2A, OUTPUT);
    pinMode(Motor1B, OUTPUT);
    pinMode(Motor2B, OUTPUT);

    digitalWrite(Motor1A, LOW);
    digitalWrite(Motor2A, LOW);
    digitalWrite(Motor1B, LOW);
    digitalWrite(Motor2B, LOW);
}

int flag1 = -1;
int flag2 = -1;

void loop() {
    if (bluetooth.available()) {
        char toSend = (char)bluetooth.read();
        if (toSend == 'S') {
            flag1 = 0;
            flag2 = 0;

            digitalWrite(Motor1A, LOW);
            digitalWrite(Motor1B, LOW);
            digitalWrite(Motor2A, LOW);
            digitalWrite(Motor2B, LOW);
        }
        if (toSend == 'F' || toSend == 'G' || toSend == 'I') {
            if (flag1 != 1) {
                // ESTOS HARAN QUE VAYA PARA ADELANTE EL CARRITO.
                flag1 = 1;
                digitalWrite(Motor1A, HIGH);
                digitalWrite(Motor1B, LOW);
                digitalWrite(Motor2A, HIGH);
                digitalWrite(Motor2B, LOW);
            }
        }
        if (toSend == 'B' || toSend == 'H' || toSend == 'J') {
            if (flag1 != 2) {
                // ESTOS HARAN LA REVERSA DEL CARRITO.
                flag1 = 2;
                digitalWrite(Motor1B, HIGH);
                digitalWrite(Motor1A, LOW);
                digitalWrite(Motor2B, HIGH);
                digitalWrite(Motor2A, LOW);
            }
        }
        if (toSend == 'L' || toSend == 'G' || toSend == 'H') {
            if (flag2 != 1) {
                // ESTOS HARAN QUE GIRE HACIA LA IZQUIERDA.
                flag2 = 1;
                digitalWrite(Motor2B, HIGH);
                digitalWrite(Motor2A, LOW);
                digitalWrite(Motor1A, HIGH);
                digitalWrite(Motor1B, LOW);
            }
        }
        else if (toSend == 'R' || toSend == 'I' || toSend == 'J') {
            if (flag2 != 2) {
                // ESTOS HARAN QUE GIRE HACIA LA DERECHA.
                flag2 = 2;
                digitalWrite(Motor1B, HIGH);
                digitalWrite(Motor1A, LOW);
                digitalWrite(Motor2A, HIGH);
                digitalWrite(Motor2B, LOW);
            }
        }
        else {
            if (flag2 != 3) {
                flag2 = 3;
                digitalWrite(Motor2A, LOW);
                digitalWrite(Motor2B, LOW);
                digitalWrite(Motor1A, LOW);
                digitalWrite(Motor1B, LOW);
            }
        }
    }
}
